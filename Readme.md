

## **Getting Started**

### **Prerequisites**
1. [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet)
2. Any IDE supporting .NET (e.g., Visual Studio, Visual Studio Code)

### **Installation**

1. Restore dependencies:
   ```bash
   dotnet restore
   ```

2. Run the application:
   ```bash
   dotnet run
   ```
